

# cell

## 组件简介

创建一个单元格，通常可以使用在列表中

## 代码片段（常用）

```vue
<gc-cell title="主标题">副标题</gc-cell>
```

## props属性

| 属性名    | 类型   | 默认值 | 说明                                 |
| --------- | ------ | ------ | ------------------------------------ |
| title     | String |        | 单元格标签                           |
| main-icon | String |        | 左侧主图标                           |
| opt-icon  | String | right  | 右侧图标                             |
| to        | String | ''     | 单元格被点击的时候需要跳转的路由地址 |


## slot

| 插槽名称 | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| ext      | 额外的显示内容                                               |
| mainIcon | 左侧主图标，如果设置了该slot会自动隐藏main-icon属性对应的图标 |
| optIcon  | 右侧操作图标，如果设置了该slot会自动隐藏opt-icon属性对应的图标 |
